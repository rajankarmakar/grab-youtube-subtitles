# Overview

Grab YouTube Subtitles makes it easy to copy and download subtitles from any YouTube video, as long as subtitles are available.

Displays subtitle timestamps and allows you to download them in TXT or SRT format.

## How to get start
1. Install extension by button “Add to Chrome”
2. Open any youtube video
3. Click button on panel of Video (Grab Subtitles button)
4. Download youtube subtitles

## Features
Formats: SRT and TXT file. <br>
⬇️ Download YouTube subtitles <br>
📋 Copy YouTube caption to clipboard

## Who Is It For?
Copy or Download subtitles from youtube for Creators, Translators, Language Learners, Students & Educators.

### Formats for downloading YouTube subs:
SRT — Subtitle file with timestamps <br>
TXT — Flexible plain text format
